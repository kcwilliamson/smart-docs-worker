Get Started with Cloudflare Workers

Cloudflare Workers allows you to deploy serverless code instantly across the globe. This guide will help you install Wrangler, the Workers CLI, and deploy your first Worker.

1Install Wrangler CLI

Wrangler is the official CLI for managing Cloudflare Workers. Install it using your preferred package manager:

üçé Installation for macOS

Recommended: Install via npm (included with Node.js)

Terminal
# Install Wrangler globally
npm install -g wrangler

# Or use Homebrew if you prefer
brew install cloudflare-wrangler2
üí° Mac Tip: If you don't have Node.js installed, we recommend using Homebrew: brew install node
ü™ü Installation for Windows

Recommended: Install via npm using PowerShell

PowerShell
# Install Wrangler globally
npm install -g wrangler

# Or use winget
winget install Cloudflare.Wrangler
üí° Windows Tip: We recommend using Windows Terminal for the best experience. Install Node.js from nodejs.org if you haven't already.
üêß Installation for Linux

Recommended: Install via npm

Bash
# Install Wrangler globally
npm install -g wrangler

# On Ubuntu/Debian, you might need to install Node.js first:
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs
üí° Linux Tip: If you encounter permission errors, avoid using sudo with npm. Instead, configure npm to use a different directory.

2Verify Installation

After installing, verify that Wrangler is installed correctly:

wrangler --version

3Authenticate with Cloudflare

Login to your Cloudflare account using Wrangler:

wrangler login

This will open your browser to complete authentication.

4Create Your First Worker

Use Wrangler to scaffold a new Worker project:

wrangler init my-worker
cd my-worker

Configuration Files

Wrangler stores its configuration in the following locations:

macOS Configuration Path: ~/.config/wrangler/config.toml
~/Library/Application Support/wrangler/
Windows Configuration Path: %APPDATA%\wrangler\config.toml
C:\Users\YourName\AppData\Roaming\wrangler\
Linux Configuration Path: ~/.config/wrangler/config.toml
$XDG_CONFIG_HOME/wrangler/

5Deploy to Production

When you're ready, deploy your Worker to Cloudflare's global network:

wrangler deploy
üéâ Congratulations! Your Worker is now running on Cloudflare's edge network in 300+ cities worldwide.
